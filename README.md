# ✈️ Airport Booking Portal (deployed)
https://airport-booking-portal.vercel.app/

## 📌 Overview
The **Airport Booking Portal** is a fully functional frontend application built using **React.js**. It simulates a real-world airport booking system with role-based access for **Passengers, Employees, and Admins**.

This project demonstrates modern frontend development practices including routing, state management, component architecture, and responsive UI design.

---

## 🚀 Features

### 🔐 Authentication
- Role-based login system (Passenger / Employee / Admin)
- Session handling using localStorage

### 👤 Passenger Portal
- Search available flights  
- Book tickets  
- View bookings  
- Cancel bookings  

### 🧑‍💼 Employee Portal
- View all bookings  
- Approve / Reject / Complete bookings  

### 🛠 Admin Portal
- Manage users (view & delete)  
- Add / delete flights  
- Dashboard with system statistics  

### 📊 Dashboard
- Dynamic stats based on role  
- Clean UI with cards and layout  

### 📄 Common Pages
- Profile management (edit user info)  
- Contact / Support page  

---

## 🎨 UI/UX Highlights
